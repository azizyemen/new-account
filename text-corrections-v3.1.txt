✅ **النقاط التسع (تم تصحيحها بالكامل):**

┌─────────────────────────────────────────────────────────────┐
│ 1️⃣ أقر بأن المعلومات المبينة مسبقاً صحيحة ومطابقة      │
│    للواقع.                                                  │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 2️⃣ أتعهد بأن البضاعة المستلمة بموجب سند استلام أو       │
│    فاتورة مبيعات موقع عليها بالاستلام مني أو من أحد      │
│    العاملين من طرفنا حتى ولو لم يكن المستلم على          │
│    مخالفاتنا أو مسؤوليتنا.                                │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 3️⃣ كما أتعهد بسداد قيمة البضاعة المستلمة طبقاً لما سبق  │
│    فور استحقاقها، كما لا تعتبر البضاعة مسددة إلا بموجب   │
│    سند قبض رسمي من شركة الخليفي للتجارة.                 │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 4️⃣ أقر وأتعهد بسداد كافة المبالغ المستحقة علينا حتى ولو  │
│    تعدت مديونيتنا الحد الائتماني المطلوب، وذلك دون شروط  │
│    حالاً عند الطلب من شركة الخليفي للتجارة، ولها الحق    │
│    في الرجوع للجهات المختصة مع مصاريف ودون اعتراض        │
│    واحتياج ويعد هذا بمثابة سند لأمر.                      │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 5️⃣ في حالة بيع المنشأة أو تغيير في الاسم التجاري أو      │
│    إيجارها أو في حالة إغلاقها لأي سبب، نتعهد بتقديم       │
│    إخطار رسمي بما يفيد ذلك.                               │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 6️⃣ ونتعهد بكوننا متضامنين شخصياً مع المالك الجديد بسداد  │
│    كافة المديونية القديمة المستحقة علينا فوراً أو وفق ما │
│    تحدده الشركة.                                           │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 7️⃣ في حالة عدم السداد خلال الفترة المحددة أو في حالة     │
│    تعدي مديونيتنا الحد الائتماني المطلوب، يحق لشركة       │
│    الخليفي للتجارة إيقاف الحساب أو تعليقه.               │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 8️⃣ عمل مطابقة للرصيد بصفة دورية مع شركة الخليفي للتجارة │
│    (حسب النظام المتبع) على أن تعتبر (مطابقة للرصد)       │
│    مصادقة وإقراراً منا بصحتها ويعتد بها أمام الجهات      │
│    المختصة أياً كانت.                                      │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 9️⃣ وبعد الموافقة على جميع ما سبق الرجاء فتح حساب خاص    │
│    بنا لديكم طبقاً للآتي:                                  │
└─────────────────────────────────────────────────────────────┘
